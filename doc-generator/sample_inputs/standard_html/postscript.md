# Redfish doc generator

This document was created using the Redfish doc generator, which uses the contents of the Redfish Schema files in JSON Schema format to automatically generate most of the text.  The source code is available for download at the DMTF's GitHub repository at <a href="http://www.github.com/DMTF/Redfish-Tools">http://www.github.com/DMTF/Redfish-Tools</a>.
